Twixer is a Chrome Extension that:

- Adds Mixer streamers to Twitch sidebar
![alt_text](assets/Screenshots/main-screenshot.png "Twixer")

- Adds Twitch sidebar to Mixer
- Adds the ability to favorite streamers
